# colored_e-gov

e-govの税法条文に色をつけます


# インストール

[Tampermonkey](http://tampermonkey.net/)を入れたブラウザで、

[colored_e-gov.js](https://m-haketa.github.io/colored_e-gov/colored_e-gov.user.js)

をクリックしてインストール。
