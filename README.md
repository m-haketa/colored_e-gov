# colored_e-gov

e-govの税法条文の括弧に色をつけるとともに、漢数字を算用数字に変換します。


# インストール

[Tampermonkey](http://tampermonkey.net/)を入れたブラウザで、

[colored_e-gov.js](https://github.com/m-haketa/colored_e-gov/raw/master/colored_e-gov.user.js)

をクリックしてインストール。
